# 📌 電腦視覺與應用專案說明

## 📝 期中專案說明  
本專案模擬基於已知結構的 3D 掃描裝置場景。在虛擬環境中，一個 3D 物體位於 200 mm 寬的立方框架內。我們假設有一個與立方體側面平行的平行光源（如太陽）。給定的影像是透過在光源前插入一根桿子來渲染的，因此投射的陰影將是物體與平面的交點，如下圖所示：
